# WaveBOX-Web-Music-Player
WaveBOX is a modern, Spotify-inspired web music player built using HTML, CSS, and Vanilla JavaScript. It provides an interactive and responsive music-streaming experience with playlist management, playback controls, and real-time audio handling , all running entirely in the browser.

Description
WaveBOX is a front-end focused music player application that demonstrates core JavaScript concepts, DOM manipulation, and media handling using the HTML5 Audio API.
The project replicates essential features of a real-world music streaming platform, such as dynamic playlists, song navigation, shuffle/repeat modes, progress tracking, and volume control, wrapped in a clean and modern UI inspired by popular music apps.
This project was built as a learning-driven portfolio project to strengthen practical understanding of frontend development and JavaScript logic.
